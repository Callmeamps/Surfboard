# Product

## Register

product

## Users

Developer and power user. Builds, experiments, iterates on browser-based workflows. Context: workstation, multiple tabs, shell commands, extending the browser with miniapps and AI sidecar.

## Product Purpose

Personal Electron browser for dev and productivity experimentation. Vertical tabs, collapsible sidebar, floating AI sidecar, browser shell, Chrome extensions, canvas pages. A fast, customizable shell that adapts to workflow.

## Brand Personality

Playful, powerful, custom.

Playful: personality in UI, not corporate sterility. Powerful: does real work. Custom: built to be modified.

## Anti-references

- **Chrome:** Generic, corporate, zero personality.
- **Edge:** Copilot-bloated, telemetry-heavy, Microsoft design.
- **Arc (caution):** Over-designed, opinionated UX that fights power users.

## Design Principles

1. **Shell over chrome.** Workspace, not product. Content-first.
2. **Modularity wins.** Every feature toggleable, extendable, replaceable.
3. **Dark by default.** Dark mode is the base, not a theme.
4. **Function over form.** Animations serve feedback, not decoration.
5. **Fast is a feature.** Instant interactions, no spinners for local ops.

## Accessibility & Inclusion

- Reduced motion: respect prefers-reduced-motion
- Dictation and TTS support
- Keyboard navigation for all features
- Screen reader semantics
- High contrast in dark theme
