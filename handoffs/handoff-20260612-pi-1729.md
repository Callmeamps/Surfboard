## Architecture

```
User clicks PDF link/navigates to .pdf URL
  → papertm.js intercepts via did-navigate / new-window
  → pdf:open(url) IPC → main process fetches + caches PDF
  → opens surfboard://pdf?id=<cacheKey> tab
  → papertm routes to PDFViewerPage.init()
  → renderer imports pdfjs-dist dynamically
  → pdf:getData(cacheKey) → ArrayBuffer
  → pdfjs renders pages to canvas
  → toolbar controls: zoom, page nav, download, print
```
