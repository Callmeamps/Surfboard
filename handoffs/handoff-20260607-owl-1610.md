### Gotchas
- `write` tool corrupts IIFE module files — use bash heredoc → `/tmp/` → `mv` pattern
- TrustManager `require()` throws, not returns false — callers must wrap in try/catch
- Jest `rejects.toThrow('exact message')` can fail for async rejections — use try/catch in test body
- Miniapps IIFE must guard `document.addEventListener` for JSDOM compatibility
