### Gotchas
- `write` tool can corrupt IIFE module files — use bash heredoc → `/tmp/` → `mv` for large IIFE files
- TrustManager `require()` throws, not returns false
- Miniapps IIFE must guard `document.addEventListener` for JSDOM test compatibility
- Playwright `elementFromPoint()` returns SVG child of buttons — use `{ force: true }` for test clicks
- `session.fromPartition()` returns a session object; in Electron 42, extension methods are at `session.extensions.loadExtension()` not `session.loadExtension()`
- `fs.Dirent.isDirectory()` returns `false` for symlinks — must check `isSymbolicLink()` separately
