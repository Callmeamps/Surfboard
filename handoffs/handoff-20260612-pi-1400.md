# Handoff 2026-06-12 pi (CI hardening session 2)

## Context

Fixed CI infrastructure bugs. Unit tests + build now pass on CI.

### Fixes Applied

1. **Checkout chicken-egg bug** (`1910cdf`) — Composite action needed `actions/checkout@v4` before `uses: ./.github/actions/setup`. Added checkout to every job in test.yml, removed from composite action.

2. **electron in dependencies** (`00253dd`) — Moved `electron` to devDependencies (electron-builder requirement).

3. **Electron/Playwright binary caching** (`00253dd`) — Added `~/.cache/electron` and `~/.cache/playwright` to CI cache.

4. **Duplicate IPC handlers** (`b7fb70f`) — `downloads:*` handlers registered in both `ipc-handlers.js` AND `download-manager.init()`. Removed from `ipc-handlers.js`. This was crashing Electron before window creation.

5. **Force X11 on Linux** (`b7fb70f`) — Wayland/Vulkan incompatible on CI. Added `--ozone-platform=x11` unless `--wayland` flag passed.

6. **E2E timeout** — Increased Playwright `firstWindow()` timeout to 60s in all e2e tests.

7. **npmRebuild: false** — Disabled native module rebuild in electron-builder (cpu-features/nan incompatible with Node 22).
