### Gotchas from this session

- The `write` tool was corrupting IIFE module files in the previous session (see `handoffs/handoff-20260607-pi.md`). Use bash heredoc → `/tmp/` → `mv` pattern for writing JS files.
- TrustManager `require()` throws, not returns false — callers must wrap in try/catch.
- Jest `rejects.toThrow('exact message')` can fail for async rejections — use try/catch in test body instead.
